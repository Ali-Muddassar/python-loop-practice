# Python Loop Practice

## Description
A comprehensive collection of Python scripts demonstrating loop structures and flow control statements. This repository covers `for` loops, `while` loops, nested loops, and loop control statements like `break`, `continue`, and `pass`. Created as part of my foundational Python learning journey with **@Codanics**.

## Topics Covered

### Loop Types
| Topic | Description |
| :--- | :--- |
| `for` loop | Iterating over sequences (lists, strings, ranges) |
| `while` loop | Executing code while a condition is true |
| Nested loops | Loops inside loops for multi-dimensional operations |
| `range()` | Generating sequences of numbers |

### Loop Control Statements
| Statement | Purpose |
| :--- | :--- |
| `break` | Exit the loop immediately |
| `continue` | Skip the current iteration and continue |
| `pass` | Placeholder that does nothing |

### Practice Includes
- Iterating through lists and strings
- Creating patterns with nested loops
- Input validation using `while` loops
- Using `range()` with start, stop, and step parameters
- Practical examples of loop control flow

## Purpose
To build a strong, practical understanding of Python loops—essential for automating repetitive tasks, processing data collections, and implementing efficient algorithms in data analysis.
